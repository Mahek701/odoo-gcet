## ğŸ”‘ Login ID Auto-Generation

When an Admin creates a new employee, the system automatically generates a Login ID using the following format:

```
[CompanyCode][First2LettersOfFirstName][First2LettersOfLastName][YearOfJoining][4-digit serial]
```

### Example:
```
OIODO20220001
```
- **OIODO**: Company code (first 5 letters, uppercase)
- **20**: First 2 letters of first name
- **22**: First 2 letters of last name
- **2022**: Year of joining
- **0001**: 4-digit serial number

The serial number increments automatically for employees joining in the same year.

## ğŸ“– Usage Guide

### For Administrators

1. **Login**: Use the default admin credentials or your created admin account
2. **Create Employees**: Click the "NEW" button on the dashboard to create employee accounts

6. **Edit Profile**: Access "My Profile" from the avatar dropdown to edit personal info and change password
